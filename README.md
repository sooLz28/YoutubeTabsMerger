# 📺 YouTube Tab Merger

A lightweight Chrome extension that finds all your open YouTube video tabs, combines them into a single temporary playlist, and closes the extra tabs to save memory.


## 📂 File Structure

Create a folder named `YT-Tab-Merger` and place these three files inside it:

1. **`manifest.json`** (The "ID card" for Chrome)
2. **`popup.html`** (The small menu interface)
3. **`popup.js`** (The logic script)

---

> **Note:** YouTube's `watch_videos` link creates a "Temporary Playlist." and it's not saved into your account.
